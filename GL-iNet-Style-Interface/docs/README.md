# GL.iNet Style Interface for OpenWrt

A modern, user-friendly web interface for OpenWrt routers inspired by GL.iNet design.

## Features
- Modern responsive design
- Real-time monitoring
- JWT authentication
- WebSocket updates
- Mobile-friendly
- Multi-language support
